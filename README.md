# Blackjack Game (Python)

A simple command-line Blackjack game built using Python using Object-Oriented Programming (OOP).

---

## 🚀 Features

* Deck and card system
* Player vs Dealer gameplay
* Blackjack rules implemented
* Ace value adjustment (1 or 11)
* Input-based user interaction

---

## 🛠️ Technologies Used

* Python
* OOP (Classes & Objects)
* Random module

---

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/your-username/blackjack-python.git
```

2. Navigate to the folder:

```
cd blackjack-python
```

3. Run the game:

```
python blackjack.py
```
